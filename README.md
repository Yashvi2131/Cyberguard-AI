# CyberGuard AI 

## Overview
CyberGuard AI X is an AI-powered cybercrime complaint management system designed to assist users in reporting cybercrime incidents and obtaining appropriate guidance. The system uses Machine Learning and Natural Language Processing techniques to analyze complaints, predict severity levels, and recommend the relevant cybercrime authority or government department.

## Key Features
* User Registration and Authentication
* Cybercrime Complaint Submission
* AI-Based Complaint Classification
* Severity Prediction
* Recommendation Engine
* Analytics Dashboard
* Login Activity Monitoring
* Complaint and Severity Logs
* National Cyber Crime Portal Guidance

## Technology Stack
* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* Machine Learning: Python, Scikit-learn
* NLP: NLTK / Transformers
